Eclipse SDK Build 20020409

What's new in this drop

API changes

  • org.eclipse.ant.core does not export org.apache.ant anymore. Plug-ins that need to have access to Ant classes need to require org.apache.ant directly.
  • Tasks refreshLocal and incrementalBuild are not available anymore. Use eclipse.refreshLocal and eclipse.incrementalBuild instead.

Other highlights

  • New property eclipse.running is set for scripts running inside Eclipse.
  • It is now possible to extend the Ant classpath by adding new JARs, folders, tasks and types through a preference (Workbench->Preferences->Ant->Customize).
