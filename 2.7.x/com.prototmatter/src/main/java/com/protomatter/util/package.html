A collection of utility classes. Error messages used as part of selected classes can be generated in different languages. Simply add a new resource (named com/protomatter/util/Resources) to your classpath for the new language. For instance, to add French error messages, you would copy the existing file com/protomatter/util/Resources.properties and make a new file called com/protomatter/util/Resources_fr.properties and place it in the CLASSPATH. Please see the JavaDoc for the java.util.ResourceBundle class for more information.
