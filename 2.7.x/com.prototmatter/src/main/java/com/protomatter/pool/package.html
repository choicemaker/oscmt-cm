An object pooling framework. Every error message shown by the pool framework can displayed in a different language. Simply add a new resource (named com/protomatter/pool/Resources) to your classpath for the new language. For instance, to add French error messages, you would copy the existing file com/protomatter/pool/Resources.properties and make a new file called com/protomatter/pool/Resources_fr.properties and place it in the CLASSPATH. Please see the JavaDoc for the java.util.ResourceBundle class for more information.
