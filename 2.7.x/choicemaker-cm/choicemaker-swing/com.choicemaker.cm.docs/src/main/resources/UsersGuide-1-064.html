1 Basic structure

The configuration file has three sections: core, plugin, and custom for the ChoiceMaker core, plugins, such as Oracle I/O, and customizations for specific customers, respectively (Figure 56). The plugin and custom sections are optional. This document focuses on the core section. The format and semantics of the other sections are described along with the respective plugins and customizations.
