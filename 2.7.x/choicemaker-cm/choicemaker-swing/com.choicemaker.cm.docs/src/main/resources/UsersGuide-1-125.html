7 Customization Options

ReviewMaker is a highly flexible program which offers many different customization options.

7.1 Customizing layouts

As mentioned above, the layout for the Review screen is modifiable in the same way as the equivalent screen in ModelMaker, as described in Chapter II.10. Layouts are loaded and saved in LayoutàReview/Search ... This same menu controls layouts for the top of the Search screen. In fact, Review and Search share the same layout, so the layout selected for one is always used by the other.

Overview and the results shown on the bottom of the Search screen, on the other hand, uses a different layout. Layouts for these screens can be loaded and saved by selecting LayoutàResults... Note that in order to add and remove columns or change the table displayed in a Results layout, you must open up the layout in the Review screen. Note that a Results layout can contain only one table, so do not load multi-table layouts in Overview and Search.

The layout that is used on these screens by default is stored on the server and is not user-modifiable, but can be easily changed by your ChoiceMaker support staff.

7.2 Viewing records in a side-by-side format

ReviewMaker and ModelMaker default to show the pair of records being reviewed with the q record on top, displayed in blue and the m record directly underneath, in green. Some users prefer to see the data displayed with one record on the right and one record on the left, however. This layout can be easily constructed in either application by inserting two copies of each relevant table and then hiding the q data on one side and the m data on the other side by dragging the center bar that separates the q from the m record.

7.3 Customizing Review Policies

By default, when the q record matches to more than one m record in Review, marking one record as Match forces all other records to be marked Differ. This, however, is merely the default policy. If you wish to adopt a different policy which allows q to match against multiple m's, ChoiceMaker Technologies can easily configure the screen to implement this policy.
