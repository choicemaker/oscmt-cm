7.5.3 Removing blocking fields

To remove a blocking field f, perform the following steps:

  1. Remove the field f from the schema (Section 7.4).
  2. Recompile the model (Chapter II.6.3).
  3. If no other blocking configuration contains f, remove all indices that contain f and are not used by any other application.
