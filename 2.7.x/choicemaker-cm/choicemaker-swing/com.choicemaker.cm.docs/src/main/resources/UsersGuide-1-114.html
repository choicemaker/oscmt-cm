8 Oracle

The Oracle plugin com.choicemaker.cm.io.db.oracle can be used for blocking in conjunction with the blocking and db plugins, for taking snapshots of record pairs, and for reading marked record pairs in conjunction with the db plugin.

In addition to the Java library, the Oracle plugin includes several database objects (DDLs and PL/SQL stored procedures).

The Oracle plugin does not have any elements of its own that can be configured through the schema. It uses the configuration elements of com.choicemaker.cm.io.db.base.
