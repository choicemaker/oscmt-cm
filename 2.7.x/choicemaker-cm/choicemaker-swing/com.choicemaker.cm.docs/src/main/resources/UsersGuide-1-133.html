1 ModelMaker

1.1 Before you begin

Before you begin the installation, review the following steps:

  1. Study and accept your license.
  2. Consult the supported platforms and make sure that your platform is supported. If necessary, install a Java 2 Platform, Standard Edition Virtual Machine.
  3. You are now ready to install ModelMaker.
1.2 Supported platforms

ModelMaker runs on any platform with a Java 2 Platform, Standard Edition Virtual Machine version 1.4 or higher and at least 128 MB of RAM and 100 MB of free disk space. The supplied installers contain JVMs for the following platforms:

  1. Microsoft Windows 98, ME, 2000, and XP.
  2. Linux for Intel x86 architecture.
  3. Sun SPARC Solaris.
  4. Sun Solaris Intel x86
  5. AIX
  6. HP-UX

For other platforms you need to first install a suitable JVM from the vendor of your operating system. Amongst others, the following UNIX systems are supported:

  1. MacOS X 10.2 and higher. Versions of MacOS X prior to 10.3 shipped with version 1.3 of Java. An update is available from Apple.
  2. Tru64 (Alpha). JDK 1.4.1 is available from Hewlett-Packard free of charge.

ModelMaker can also be installed on non-UNIX systems with JVM 1.4 or higher using the Java installer.

1.3 Installing ModelMaker
  1. If you have a previous version of ModelMaker installed, it is recommended that you first uninstall that version and delete any remaining directories and files that you no longer need.
  2. To install ModelMaker, insert the CD into a CD-ROM drive on you system. Open the CD, browse to ModelMaker, and then to your platform. Double click or launch the installer from a command shell. On platforms for which no specific installer exists and on which the generic UNIX installer cannot be used, the Java installer can be launched by executing java -jar install.jar in a command shell.
  3. Follow the on-screen instructions.
    1. Read the Introduction, and press Next.
    2. Choose the installation folder, and press Next. ChoiceMaker can run from both local and network drives.
    3. Choose the desired option for shortcuts, and press Next.
    4. Choose a Java Virtual Machine, and press Next. Remember that ModelMaker requires a version 1.4 or higher virtual machine.
    5. Double check the selected options in the Pre-Installation Summary, and press Install.
    6. The installation is complete. Click Done to exit the installer
  4. If your installer did not already include a license file, copy your license file to ChoiceMaker\com.choicemaker.cm.core\lib.
