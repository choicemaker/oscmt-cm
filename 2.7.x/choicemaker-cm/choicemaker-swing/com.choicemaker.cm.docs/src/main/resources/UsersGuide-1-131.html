3 ClueMaker compiler

This section details the limitations of the current implementation of the ClueMaker compiler:

  1. No quantifiers, shorthand form, valid, or let expressions are allowed inside shorthand forms.
  2. The ClueMaker compiler does not check the body of embedded Java methods. Errors in them will be signaled by the Java compiler.
  3. Nested classes are not permitted in ClueMaker.
  4. It is not possible to access nested Java classes in ClueMaker.
  5. Multi-clues (foreach) cannot be referenced in other clues.
  6. Only boolean and int clue sets are supported.
  7. The ClueMaker compiler only performs very limited optimizations.
