9.4.1 Probability Histogram

The probability histogram shows the record pairs based on the probability (Figure 16). The color of the bars represents the human-marked decision.

The user may change the histogram bin width by entering a new value in the field below the graph. Clicking on any bin of the probability histogram sets the record pair filter and displays the indices of the corresponding pairs in the pair list. A right click on a bin brings up a context menu that also allows all of the above choices and the selection of all pairs in a given probability range.
