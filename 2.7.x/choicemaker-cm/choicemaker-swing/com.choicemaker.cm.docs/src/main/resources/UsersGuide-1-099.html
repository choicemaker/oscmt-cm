5.1.2 Examples of record sources

Record sources are like marked record pair sources, except that there is no pair descriptor, that records do not come in pairs, and that there is no `one line per pair' option. E.g., the following is an example of non-stacked pipe-delimited, untagged data in a single file:

MARTIN|BUECHI|1735 FIRST AVE|NEW YORK|NY

ANDREW|BORTHWICK|41 EAST 11TH STREET|NEW YORK NY

JIM|MILLER|41 EAST 13TH STREET|NEW YORK NY

...
