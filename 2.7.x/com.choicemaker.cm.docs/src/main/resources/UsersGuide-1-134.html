ReviewMaker is installed using Java WebStart over the Web. The installer can be deployed as a zip file onto any Web server or as a Java WAR/EAR archive onto a Servlet container. Detailed installation instructions are provided separately.

For platforms that do not support Java WebStart,1 an InstallAnywhere installer similar to the one for ModelMaker (Section 1) is available.

1Currently Microsoft Windows, Linux on Intel x86, MacOS X, and Sun Solaris support Java WebStart.
