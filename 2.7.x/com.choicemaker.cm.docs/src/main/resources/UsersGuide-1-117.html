Chapter X. ChoiceMaker Server

1 Introduction

ChoiceMaker server provides for real-time matching against a database and serves human review cases to ReviewMaker or a custom human review GUI. ChoiceMaker Server is available in two versions:

  • ChoiceMaker Server J2EE is a J2EE 1.3 compliant component with an EJB interface. A WebService (SOAP) façade is available for select J2EE servers and will be available for all J2EE 1.4 compliant servers.
  • ChoiceMaker Server Standalone is a standalone Java RMI or CORBA server assembled to the specific requirements of a customer. Documentation for this product is provided separately.

The remainder of this chapter focuses on ChoiceMaker Server J2EE.
