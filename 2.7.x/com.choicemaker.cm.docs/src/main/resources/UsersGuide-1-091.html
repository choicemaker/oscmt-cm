4.1.1 Marked record pair sources

Marked record pair sources have ChoiceMakerMarkedRecordPairs as root element. Each pair is inside a MarkedRecordPair element. The latter has the following mandatory attributes:

Name

Semantics

Use

decision

The human decision for this pair. One of:

  • differ
  • hold
  • match

required

date

The date on which the human decision was made.

required

user

The user who made the decision.

required

src

The source of this pair.

required

comment

A comment.

required

The following is a legal marked record pair source for the schema StackedSchema.schema (Figure 32):

<?xml version="1.0" encoding="UTF-8"?>

<ChoiceMakerMarkedRecordPairs>

<MarkedRecordPair decision="match" date="2/1/2002" user="rjohnson" src="INITIAL1" comment="">

<person personId="21" firstName="JIM" lastName="SMITH">

<addr address="41 E 11th Street" lastDateReported="01/23/2003"/>

<addr address="51 E 11th"/>

</person>

<person personId="23" firstName="JAMES" lastName="SMITH">

<addr address="41 East 11th St"/>

</person>

</MarkedRecordPair>

<MarkedRecordPair decision="differ" date="2/1/2002" user="rjohnson" src="INITIAL1" comment="">

<person personId="25" firstName="UNKNOWN" />

<person personId="28" firstName="JOHN" lastName="SMITH">

<addr address="41 East 11th St"/>

</person>

</MarkedRecordPair>

...

</ChoiceMakerMarkedRecordPairs>
