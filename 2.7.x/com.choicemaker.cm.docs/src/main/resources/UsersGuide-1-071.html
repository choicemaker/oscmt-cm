2 String manipulation

The class com.choicemaker.cm.core.util.StringUtils offers several public static methods that are useful for matching:

Method

Semantics

boolean nonEmptyString(String s)

Returns true if s is non-null and contains any non white space characters.

char getChar(String s)

Returns the first character of s or `\0' if s is null or zero-length.

int countDigits(String s)

Returns the number of digits in s.

boolean overlap(String s1, String s2)

Returns true if one string begins or ends with the other.

String removePunctuation(String s)

Replaces '.', ';' and ',' with spaces and simply removes "'". Does not create leading spaces. Removes all trailing spaces.

String removeApostrophies(String s)

Removes "'".
