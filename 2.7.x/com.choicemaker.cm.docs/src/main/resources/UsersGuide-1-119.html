Chapter XI. ReviewMaker

1 Introduction

ReviewMaker is ChoiceMaker's human review and interactive query tool. Clients need to review record pairs in ModelMaker when the ChoiceMaker decision-making engine is unable to make a "match" or "differ" decision within the client's tolerance for error. The decision-making engine can send records to human review either because probabilistic scoring assigns them a probability greater than the differ threshold, but lower than the match threshold or because a rule fires which forces a hold decision.

ReviewMaker offers three basic modes of operation or views. The "Search" view allows the user to search for a record in the human-review queue. The "Overview" view provides an overview of a batch of up to 100 queued records and allows the user to select an individual record from the list. Selecting a record in either view brings the record up in the "Review" view. In Review, the ChoiceMaker server dynamically queries the database to find records which might match the record being reviewed. The review view then allows the user to mark the queued record as a match, differ, or hold to all of the records retrieved by the server.

ReviewMaker is easily installed over the web via Java WebStart. Java WebStart also enables ReviewMaker to be automatically updated over the Web.
