6 Component architecture

ChoiceMaker has a component-based architecture. Support for different input and output formats, such as XML, flat files, or Oracle, as well as for different machine learning techniques is provided through plugins. This allows a customer to license only needed components and to assemble a solution with a minimal memory footprint.

In addition, ChoiceMaker can easily and at a low cost provide additional functionality based on customer requests.
