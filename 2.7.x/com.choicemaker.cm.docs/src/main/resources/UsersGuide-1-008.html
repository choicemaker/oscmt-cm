3 Scoring

Scoring is the second pass of matching. Each potentially matching record returned by blocking is compared individually against the query record. For each record, a decision of differ, hold, or match is made.

Scoring itself is done in two steps. In the first step, a probabilistic decision is made using a machine learning technique. In the second step, rules may override the probabilistic decision. The flow, which is the same for each record returned by blocking, is illustrated in Figure 2.

3.1 Probabilistic decision making

Probabilistic decision making is the first step of scoring. ChoiceMaker uses sophisticated artificial intelligence technology in this step to arrive at highly accurate decisions.

The query record and the potential match record are compared according to different criteria. For example:

  • Are the first names the same and are they common/uncommon/very rare.
  • Do the last names have the same phoneticization according to Soundex or similar techniques?
  • Is the date of birth different?

Each criteria gives a clue as to whether the two records denote the same physical entity, e.g., the same person, or not. A machine learning technique is used to weight all the possibly contradictory clues and produce a probabilistic decision.
