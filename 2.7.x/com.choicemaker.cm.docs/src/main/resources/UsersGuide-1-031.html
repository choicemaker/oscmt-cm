9.4.4 Calculator

The calculator allows the calculation of several statistical values based on one or two input values, including false negatives, differ threshold, differ recall, false positives, match threshold, match recall, and human review.

The buttons in the output panel allow the user to easily set one or both of the current thresholds to the computed values.
