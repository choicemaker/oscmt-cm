1 About this guide

The ChoiceMaker User's Guide serves as a guide and reference to the ChoiceMaker product suite. It is targeted at technical users who deploy ChoiceMaker. It covers the customization for a specific database.

This User's Guide is organized as follows:

  • Chapter I, "Introduction," gives an overview of approximate record matching, its applications, and ChoiceMaker's approach.
  • Chapter II, "ModelMaker," describes ModelMaker, ChoiceMaker's integrated development environment for the customization of record matching to a particular database.
  • Chapter III, "ChoiceMaker schemas," defines ChoiceMaker's description of metadata.
  • Chapter IV, "Clues and ClueMaker," introduces ChoiceMaker's programming language for record comparison.
  • Chapter V, "Configuration," details the configuration of ChoiceMaker.
  • Chapter VI, "Matching support," gives an overview of the provided matching libraries, which include approximate string matching and name and address standardization.
  • Chapter VII, "Plugins," introduces ChoiceMaker's component architecture.
  • Chapter VIII, "Machine learning plugins," describes the plugins that provide the different machine learning techniques, such as maximum entropy and support vector machines.
  • Chapter IX, "Input/output plugins," gives an overview of the different input/output plugins, including support for flat files, XML, and databases.
  • Appendix A, "Limitations of the current implementation," describes the known limitations of the current implementation.

A few advanced topics, such as the creation of new plugins, are treated in the separate Developer's Guide.
