7.3.5 Illegal combinations

It is possible to explicitly prohibit a set of fields from being used as a blocking set or as a part thereof. For example, we might have both zip and city as blocking fields, but we might not want a blocking set to contain conditions on these two fields. Illegal combinations can be defined in terms of source, target, and blocking fields, or any combination thereof.
