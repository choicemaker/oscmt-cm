2 Directory structure

Figure 58 shows the overall directory structure of ChoiceMaker.

All plugins have a similar internal directory structure. Some directories may not be present in all components. Figure 59 illustrates the typical directory structure. Directories containing only generated data are in italics.

The separation of source branches makes it easy to compile only certain branches, e.g., don't compile the tests and to compile to different targets, which is necessary to support selective reloading in ModelMaker.
