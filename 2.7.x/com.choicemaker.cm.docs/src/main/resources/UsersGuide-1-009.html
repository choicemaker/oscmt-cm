3.1.1 Clues

Clues are written in ChoiceMaker's Java-based customization language called ClueMaker. For instance, a clue which predicts match given a match between the Soundex codes of the first names can be added with the following code:

clue mSoundexFirst {
match same(Soundex.soundex(r.first_name));
}

A given clue is said to fire on a record pair when all the conditions for the application of that clue on the pair are met, e.g., if the first names on the two records have the same Soundex phoneticization.

ClueMaker allows access to any Java library, includes powerful constructs to handle stacked data, and is easy to learn for Java, C, and C++ programmers. ClueMaker is described in detail in Chapter IV.
