11 Summary

ChoiceMaker schemas define the type of records matched by ChoiceMaker and how this data is exchanged with clients and databases. Generators create customized holder and translator classes, which yields high run-time speed.

ChoiceMaker schemas are XML documents, which allows us to write generators using standard XML parsing APIs. Furthermore, XML tools can be used to handle ChoiceMaker schemas.

The plug-in interface for translator generators and the XML Schema based document type definition make translator generators true components that can be selected and composed for specific deployments of ChoiceMaker.
