6 Report

The modifiers note and report can be placed before the keywords clue and rule (Section 7). The name and or the number of the clue will then be reported back to the caller by the ChoiceMaker façade. The modifier report differs from note in that it forces a match record to be returned to the caller, even if its decision is "differ".

Note, though, that this does not affect blocking. E.g., a record that is considered to differ by blocking will never be returned even if it would meet the clue or rule's predicate.
