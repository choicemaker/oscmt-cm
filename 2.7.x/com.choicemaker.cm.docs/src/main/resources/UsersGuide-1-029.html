9.4.2 Hold percentage vs. accuracy

This diagram plots the hold percentage vs accuracy (Figure 17). The horizontal axis represents the accuracy. The vertical axis represents the percent of record pairs assigned as hold by ChoiceMaker. The differ and match accuracies are equivalent at all points. Note that the differ and match thresholds independently determine the differ and match accuracies, respectively. For a given accuracy, this plot uses match and differ thresholds so the differ accuracy and match accuracy both equal the given accuracy.

On the right hand side of the graph, the accuracy, human review percentage, differ, and match thresholds are shown in two tables. The upper table lists these values for a set of fixed accuracies, the lower for a set of fixed human review percentages. The default values can be overridden in the configuration file.

Clicking in a cell of the first two columns sets both thresholds. Clicking in a cell of the differ or match threshold column sets the respective threshold.

The heights of the two tables can be adjusted by dragging the divider.
