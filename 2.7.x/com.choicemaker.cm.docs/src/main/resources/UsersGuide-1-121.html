ReviewMaker is a client-server application. Users interact with the application, which Java WebStart installs on their desktop, while the data resides on the ReviewMaker server. Therefore, in order to access any data in ReviewMaker, the user needs to log on to the server. This is done by choosing File->Login... which brings up a the simple login dialog shown in requesting your user name and password. Enter the user name and password that were assigned to you by your system administrator and click "Login". One other entry is labeled "Service" and contains the URI of the server that you will be accessing. Leave this field at its default value unless told otherwise by your system administrator.

Only one instance of ReviewMaker can be running on your computer at onceany time.
