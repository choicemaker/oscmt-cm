2 Display

The elements displayNodeType and displayField are used to define how data is displayed in the review screens of ModelMaker (Chapter II.10) and ReviewMaker.

The element displayNodeType has the following attribute:

Name

Semantics

Use

Default

name

The name to be displayed for the node type. If not specified, the value of name in nodeType is used.

optional

N/A

The element displayField has the following attributes:

Name

Semantics

Use

Default

name

The name to be displayed for field. If not specified, the value of name in field is used.

optional

N/A

use

Whether the field should be displayed.

optional

true
