3 Name and file name

A ChoiceMaker schema is an XML document that declares the type of the records to match. Figure 31 gives an example of a simple schema. Line 1 contains the standard XML header declaration.1

Line 2 defines the name of the schema to be SimpleSchema. The schema name and its file name must be the same with the file extension `.schema'. In other words, the figure shows the content of the file SimpleSchema.schema.

Lines 3 and 4 list the import declarations that correspond to their counterparts in Java classes. An import declaration allows a named type to be referred to by a simple name that consists of a single identifier. As in Java, the package java.lang is implicitly imported.

1Encodings other than UTF-8, e.g., ShiftJIS, are permitted. However, there are character set restrictions for certain identifiers.
