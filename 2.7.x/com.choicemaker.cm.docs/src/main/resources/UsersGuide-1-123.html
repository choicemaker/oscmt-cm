5 The Search View

The search view allows the user to search the queue for records. To initiate a search, simply click on the appropriate field, type in the value that you want to search for, and click "Search".

shows a search for individuals in the queue with the last name of Abreu. ReviewMaker uses the ChoiceMaker matching engine to perform the search, so multiple fields can be used in the search. shows a search on the same last name, but with approximate matches on the first name and date. Note that the second and third records from the "last name only" search are excluded. The "Reset Query" button resets all the fields in the current query.

The lower half of the Search screen is identical to the Overview screen, so all the buttons work the same as described above. If a search returns more than 100 records, "Next" and "Previous" allow the user to view the records 100 at a time. Clicking "Review Selection" and double-clicking a record bring up selected record in the Review view, and "Refresh" refreshes the list.
