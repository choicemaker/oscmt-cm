Chapter VII. Plugins

1 Introduction

This User's Guide describes the standard plugins that ship with ChoiceMaker. Your version may contain additional plugins and not contain some of the plugins listed here.

ChoiceMaker uses the Eclipse plugin framework. The following list shows a few sample plugins:

  • com.choicemaker.cm.core: The ChoiceMaker core classes, ClueMaker compiler, and schema generator.
  • com.choicemaker.cm.modelmaker: The ModelMaker GUI.
  • com.choicemaker.cm.io.xml.base: Support for XML I/O.
  • com.choicemaker.cm.io.xml.gui: GUI for configuring XML I/O in ModelMaker.
  • com.choicemaker.cm.ml.me.base: Maximum entropy machine learning.
  • com.choicemaker.cm.ml.me.gui: GUI for configuring maximum entropy machine learning in ModelMaker.
  • com.choicemaker.cm.server.base: ChoiceMaker server.
  • com.choicemaker.cm.reviewmaker.gui: ReviewMaker human review GUI.
  • com.choicemaker.cm.custom.x: Custom plugin created for customer x.
