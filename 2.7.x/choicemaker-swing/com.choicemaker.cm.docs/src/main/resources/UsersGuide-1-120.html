2 Installation

ReviewMaker can be accessed at http://<host>:8080/ReviewMaker/. Your ChoiceMaker system administrator will tell you the name of the host. With Internet Explorer on Microsoft Windows 98/2000/NT/XP, the provided link will automatically install Java WebStart if it is not yet present. Automatic installation has been shown to fail on certain machines. In this case, Java WebStart must be installed manually using either the JRE or JDK installers provided by Sun.

On all other operating systems, Java WebStart must be installed first.

The second time that you launch ReviewMaker, you will be asked whether you want to install a ReviewMaker icon on your desktop, in your start menu, or both. This will allow you to launch ReviewMaker as you would any other application.

Whenever you launch ReviewMaker, Java WebStart will check whether any ReviewMaker client updates have been installed on the ReviewMaker server. If so, they will be automatically installed on your machine.
