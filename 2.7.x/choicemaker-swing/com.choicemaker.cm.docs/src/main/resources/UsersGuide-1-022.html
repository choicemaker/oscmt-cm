5 Error Dialog

The error dialog (Figure 8) displays errors that occur during ModelMaker's execution. The Less/More button toggles whether the error detail is shown. Usually, this detail is simply the stack trace from the thrown Java Exception. The Copy to Clipboard button copies the error detail to the system clipboard, so the user can send an error report to ChoiceMaker Technologies. The Help button shows a relevant help dialog, and the Close button hides the error dialog and causes ModelMaker to continue its execution.
