2 Conventions

The following conventions are used in this guide.

2.1 Typographical conventions

Typewriter font

Typewriter font is used for the following items:
  • Shell commands and arguments that you type

Sans-serif font

Narrow sans-serif font

Sans-serif font is used for the following items:
  • ClueMaker source code
  • ChoiceMaker schemas (XML)
  • ChoiceMaker configuration files (XML)
  • Other XML files
  • Examples of file contents

Italic Text

Italics are used for the following items
  • Path names and file names
  • Titles of external documents
  • Terms introduced for the first time
  • Sample values

Bold Text

Bold is used for the following items:
  • Controls in procedures, such as menu items, button names, and keyboard key names
  • Email addresses and URLs

<value>

Placeholder for a value that you must supply.

2.2 Other conventions

Menu->Item

Select a menu and choose a menu item

path\file

Directory paths use backslashes. For Unix and Linux systems, substitute forward slashes. In the classpaths of ChoiceMaker configuration files and ant build files the two can be used interchangeably.

path;path

Semicolons are used as separators for classpaths. For Unix and Linux systems, substitute colons. In the classpaths of ChoiceMaker configuration files and ant build files the two can be used interchangeably.

screen shots

All screen shots were taken on Windows 2000 with the default settings. Some elements may look slightly different on your computer.

2.3 Installing ChoiceMaker

ChoiceMaker ModelMaker is delivered as a single installer file that can be installed by double-clicking it on most platforms.

ChoiceMaker Server can be delivered as installer or as zip archive. You receive special installation instructions for your case.
