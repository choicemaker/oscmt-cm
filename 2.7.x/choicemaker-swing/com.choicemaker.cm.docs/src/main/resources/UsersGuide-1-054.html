5 Java methods

ClueMaker allows any side-effect free Java method to be defined and used in a clue set. Examples include Soundex.soundex(lastName) from ChoiceMaker's matching library with argument lastName, addr.lastDateReported.getTime() from the JDK (java.util.Date) invoked on lastDateReported, and the hypothetical Japanese.soundAlikeKanjis(q.firstName, m.firstName).

In some cases, it is convenient to define additional matching methods as part of the clue set rather than having to define a new Java class and import it. Figure 52 shows an example of a clue set with an embedded Java method. The sample clue mFilteredUpperFirstName uses string concatenation (`+') and contains calls to the JDK method toUpperCase (conversion of all characters to upper case) of class String and to the embedded Java method filterNonCharacters.
