3 Dates

ChoiceMaker provides two classes to handle dates. The class com.choicemaker.cm.core.util.DateHelper provides the following public static methods:

Method

Semantics

Date parseSqlTimestamp(String s)

Parse a date that is in the following format (java.sql.Timestamp.toString()):

yyyy-mm-dd hh:mm:ss.fffffffff

Date parseSqlDate(String s)

Parse a date that is in the following format (java.sql.Date.toString()):

yyyy-mm-dd

Date parseSqlDateOrTimestamp(String s)

Parse a date that is in one of the two above formats.

String format(Date d)

Formats the date according to the current locale's SHORT format (see java.text.DateFormat).

String formatDb(Date d)

Format the date in the following format:

yyyy-MMM-dd, e.g., 2003-JAN-11

int numbersOnly(Date d)

Returns the date as number according to the following formula:

year + 10000 * day + x * month, where x is 100000 if day < 10 and 100000 otherwise

int numericFormat(Date d)

Returns the date as number according to the following formula:

day + 100 * month + 10000 * year

boolean missingNumber(Date d1,

Date d2)

Returns true if the year of d1 and d2 are the same and:

  • the months are the same and the day of one is the day of the other with a digit added on the left or right
  • the days are the same and the month of one is the month of the other with a digit added on the left or right

For example, the function returns true for `1/5/2002' and `1/25/2002'.

float daysApart(Date date1, Date date2)

The difference between d1 and d2 in days.

boolean sameMonthAndYear(Date d1,

Date d2)

Returns whether the month and year of d1 and d2 are the same.

boolean sameDayAndMonth(Date d1,

Date d2)

Returns whether the day and month of d1 and d2 are the same.

boolean swappedDayMonth(Date d1,

Date d2)

Returns whether the year of d1 and d2 are the same and the day and month are swapped.
