Chapter V. Configuration

ChoiceMaker stores all configuration information in a single XML-based configuration file. A user can create multiple configuration files and select the one to be used through a command line parameter or, for ModelMaker, through a dialog.

The configuration file determines the use of additional classpaths, configuration of production models, etc. By default, the configuration file is called project.xml and resides in the project directory.

ChoiceMaker Technologies delivers ChoiceMaker with a configuration file appropriate for your situation. This chapter will assist you in modifying this file.
