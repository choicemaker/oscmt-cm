Appendix A. Limitations of the current implementation

1 ModelMaker

This section details the limitations of the current implementation of ModelMaker:

  1. If nohold rules are used, all values on the Test panel may be wrong.
  2. If a run-time report file is used as XML marked record pair source and the data of the base record is modified in the Review panel, the probabilities, decision, and active clues for other pairs with the same base record are not recomputed automatically.
