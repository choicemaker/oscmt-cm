7.2.1 Composite

A composite source (Figure 12) is a combination of one or more other sources. The composite source contains a list of references (names) of constituent sources that make up the composite source. Hence, a composite source can be used to `combine' multiple source of the same or different source types without the need to create a single file or database that contains all the record data.

The location of the descriptors of the constituent sources can be stored either relative to the composite descriptor or as an absolute path. If a location is stored relative to the composite descriptor, copying both the composite and the constituent descriptors to a different location works seamlessly. Using an absolute path, the composite descriptor by itself can be copied around while the constituent descriptor stays in the same directory. A relative path is the default. Note that the choice is governed exclusively by the drop down selector. E.g., if Relative to Composite Descriptor is selected, the location is stored as a relative path even if the user enters an absolute path, manually or through the file chooser.

Note that the deletion or renaming of a constituent source breaks the composite source.

7.3 Opening an existing source

An existing source can be opened with Source->Training->Open... and Source->Testing->Open..., respectively, or by choosing one of the most recently used sources from the shortcut list at the end of the Source->Training or Source->Testing menu. The properties that were set upon creation of the source can be modified with Source-> Training->Edit... and Source-> Testing->Edit..... Source descriptors and XML sources can be opened.

7.4 Holds

The user can choose whether record pairs that have been marked hold should be used or not by checking Source->Training->Include holds and Source->Testing->Include holds. If this is not checked, all records marked hold will be hidden in all three panels. Note that the record pair indices, as displayed in the pair list and on the review panel, change if holds are included.
