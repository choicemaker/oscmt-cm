5 ChoiceMaker software

ChoiceMaker's software consists of two components: ModelMaker and the ChoiceMaker server.

5.1 ModelMaker

ModelMaker is used for creating and testing probability models. ModelMaker is described in detail in Chapter II.

5.2 Server

The ChoiceMaker server is the runtime matching engine. It can be deployed as a multi-threaded Java library, an Enterprise Java Bean, a CORBA or RMI server, or a Microsoft COM component. The ChoiceMaker server is customized according to the customer's needs.

In addition to the core matching component, ChoiceMaker can be extended to perform linking and merging itself.
