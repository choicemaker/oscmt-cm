9 Microsoft SQL Server 2000

The Microsoft SQL Server 2000 plugin com.choicemaker.cm.io.db.sqlserver can be used for blocking in conjunction with the blocking and database plugins.

In addition to the Java library, the SQL Server plugin includes several database objects (tables).

9.1 Configuration

The SQL Server plugin does not have any elements that can be configured through the schema. Data sources must be configured through the connection pool of the db plugin or through JNDI.
